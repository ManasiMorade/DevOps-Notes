Day 2 - Git & GitHub

Types of Repository

1. Public Repository
    * Anyone can access and view the repository.
2. Private Repository
    * Only authorized users can access the repository.

Creating a GitHub Account

* Sign up using Email ID.
* Create Password.
* Verify Email.
* Complete Profile Setup.

Creating a Repository

1. Click on New Repository.
2. Enter Repository Name.
3. Add Description (Optional).
4. Select Visibility (Public/Private).
5. Add README File.
6. Add .gitignore (Optional).
7. Choose License (Optional).
8. Click Create Repository
